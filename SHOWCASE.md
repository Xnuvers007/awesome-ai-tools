# Showcase — 10 Cool AI Tools (Kurasi singkat)

Berikut 10 tools & projek menarik yang sering digunakan dalam ekosistem AI modern. Ringkasan cepat untuk pengunjung yang ingin "yang terbaik" di tiap kategori.

1. LangChain — Framework untuk membangun aplikasi LLM (chains, agents, connectors). Cocok untuk RAG dan orchestration. https://github.com/langchain-ai/langchain
2. LlamaIndex (sebelumnya GPT-Index) — Indexing & retrieval untuk membangun knowledge-based agents. Bagus untuk dokumen besar. https://github.com/jerryjliu/llama_index
3. Qdrant — Vector database yang dioptimalkan untuk similarity search, mudah diintegrasikan dengan banyak stack. https://qdrant.tech
4. Chroma — Lightweight local vector DB, bagus untuk prototyping dan self-host. https://www.trychroma.com
5. Hugging Face Hub — Model hub besar untuk model, tokenizer, dataset, dan spaces. Sumber utama model open-source. https://huggingface.co
6. Mistral / Mistral-7B — Contoh model open-source berperforma tinggi (cek lisensi & T&C sebelum penggunaan komersial). https://www.mistral.ai
7. Gradio — Build UI interaktif dengan cepat untuk demo model (integrasi dengan HF Spaces). https://gradio.app
8. BentoML — Deployment toolkit untuk packaging dan serve model ke production. https://www.bentoml.com
9. Whisper / WhisperX — Automatic Speech Recognition (ASR) yang kuat untuk transkripsi audio. https://github.com/openai/whisper
10. Evals (OpenAI / HF Evaluate / LM-eval-harness) — Tooling untuk evaluasi model dan benchmark performa. Gunakan untuk memastikan model aman dan akurat.

Tips: untuk setiap tool di daftar utama Anda, tambahkan catatan singkat "why use it" dan label seperti `self-host` / `paid`. Itu membantu pengunjung cepat memutuskan.
